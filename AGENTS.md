# Web Serial Port

Portable tool to provide a WebUI for Serial Ports. Backend is RUST and FrontEnd is NextJS/ReactJS with SQLite for storage. 

- User can select and open and close serial ports. Select BAUD Rates.
- Change Display coding from ASCII to HEX. Send data buffers as HEX or ASCII.
- User can Start and Stop Capture/Logging
- User can create Send Buffer Macros
